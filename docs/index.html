lathecode

Text format for lathe parts and other objects with circular symmetry, see GitHub.

Smallest possible lathecode file describing 7mm long 6mm pin can be read as "Length 7mm, Diameter 6mm":

L7 D6

lathecode to cut same pin from 10mm stock with a 3mm wide parting blade:

STOCK R5
TOOL RECT L3
L7 R3
L3

To make a cone, specify start and end diameter or radius. MT2 to B16 arbor can be specified like:

L80 DS14.9 DE17.78
L2 D15.733 ; spacer
L24 DS15.733 DE14.5

Early demo of an editor that visualizes lathecode and generates GCode for it.
